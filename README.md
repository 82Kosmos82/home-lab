# Home Lab

Особиста лабораторія для поглиблення знань з системного адміністрування
Шлях від базової AD-інфраструктури до хмарних сервісів з вивченням елементів автоматизації.

## Мета

Прокачка навичок для працевлаштування на позицію системного адміністратора після звільнення з армії.

## Архітектура

> Схема буде додана після Етапу 1.

1. **Все автоматизується.** Що зроблено мишкою — повторюється скриптом.
2. **Все документується.** Кожен етап має README з контекстом.
3. **Все версіонується.** Git як єдине джерело правди.
4. **Безпека першочергова.** Жодних секретів у репозиторії.
